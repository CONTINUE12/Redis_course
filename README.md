# Redis_course

#### 介绍

Java技术栈之Redis学习

#### 概念

Remote Dictionary Server（远程字典服务），是一个开源的可基于内存亦可持久化的日志型、Key-Value数据库。

#### 作用

可用作高速缓存，也可用作数据库。
